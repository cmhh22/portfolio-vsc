# Portfolio VSC

Proyecto de portafolio para mostrar trabajos. Estructura mínima:

- `index.html` — entrada principal
- `app.jsx`, `sections.jsx` — código frontend
- `styles.css` — estilos
- `api/chat.js` — función serverless

Para probar localmente, abrir `index.html` en el navegador o usar un servidor estático.

Deploy recomendado: Vercel (archivo `vercel.json` incluido).
